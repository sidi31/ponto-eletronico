# Sistema de Ponto Eletrônico

Projeto simples de ponto eletrônico feito com HTML, CSS e JavaScript puro.

Os dados são salvos no `localStorage` do navegador.

## Como executar

Abra o arquivo `index.html` no navegador.

## Estrutura

```text
/
+-- index.html
+-- pages/
|   +-- registro.html
|   +-- confirmado.html
|   +-- historico.html
|   +-- perfil.html
|   +-- admin.html
|   +-- admin-servidor.html
|   +-- admin-relatorios.html
|   +-- login-admin.html
+-- assets/
|   +-- css/
|   |   +-- estilo.css
|   +-- js/
|   |   +-- script.js
|   |   +-- storage.js
|   +-- img/
|       +-- flag_Santiago.png
|       +-- image_sem_fundo.png
+-- data/
|   +-- usuarios.json
|   +-- registros.json
+-- docs/
    +-- PROJETO.md
```

## Documentação

A proposta resumida do projeto está em `docs/PROJETO.md`.

## Observação

Esta versão é demonstrativa. Não usa backend, banco de dados externo, Node.js ou bibliotecas.
